2、服务器IOMSrv工作任务

官方也提供了zabbix安装的文档，也可以自己按照官方文档进行安装，文档的链接如下：

https://www.zabbix.com/download?zabbix=5.0&os_distribution=centos&os_version=7&db=mysql&ws=nginx

1、版本选择

2、安装Zabbix部署包

[root@imosrv ~]# yum install zabbix-server-mysql zabbix-web-mysql zabbix-agent

3、后端存储数据库

在MySQL上安装Zabbix数据库和用户，请参看下列指导步骤。MySQL数据库创建脚本。

[root@storagesrv ~]# mysql -uroot -pChinaskill21!
create database zabbix character set utf8 collate utf8_bin;
grant all privileges on zabbix.* to zabbix@'%' identified by 'Chinaskill21!';
flush privileges;

然后导入初始架构（Schema）和数据。

zcat /usr/share/doc/zabbix-server-mysql-*/create.sql.gz | mysql -uzabbix -pChinaskill21! zabbix

4、编辑配置文件

Zabbix Server端配置文件

[root@imosrv ~]# vi /etc/zabbix/zabbix_server.conf
DBHost=192.168.100.200
DBName=zabbix
DBUser=zabbix
DBPassword=Chinaskill21!
#DBPort=3358

Zabbix Agent端配置文件

# 自己监控自己。。。
[root@imosrv ~]# vim /etc/zabbix/zabbix_agentd.conf
Server=192.168.100.110 # Zabbix Server的地址，允许该ip地址获取数据，被动获取
ServerActive=192.168.100.110 # 主动将主机的数据发送给该ip地址，Zabbix Server，主动发送
Hostname=IOMSrv # 这个定义的名字必须和web页面里面host的名字一样。
UnsafeUserParameters=1  #允许所有的字符是在用户定义的参数，参数传递，也就是支持自定义脚本

5、启动Zabbix

[root@imosrv ~]#  systemctl enable --now zabbix-server zabbix-agent

6、添加操作系统监控对象

通过Linux代理模板，添加StorageSrv、AppSrv操作系统监控对象，查看运行状态

[root@storagesrv ~]# yum install zabbix-agent vim -y
[root@storagesrv ~]# vim /etc/zabbix/zabbix_agentd.conf
Server=192.168.100.110
ServerActive=192.168.100.110
Hostname=StorageSrv
